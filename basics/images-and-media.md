# Images & media

## Adding visual content

Enhance your documentation with images, videos, and other media.

### Images

Drag and drop images directly into the editor, or use Markdown:

```markdown
![Description](path/to/image.png)
```

### Supported formats

- **Images**: PNG, JPG, GIF, SVG, WebP
- **Videos**: YouTube, Vimeo, Loom embeds
- **Files**: PDF, ZIP, and other downloadable files

### Image sizing

Control image size with HTML:

```html
<img src="image.png" width="300" />
```

### Embedding videos

Use the `/embed` command and paste your video URL:

- YouTube links
- Vimeo links
- Loom recordings

### Best practices

1. Use descriptive alt text for accessibility
2. Optimize images for web (compress large files)
3. Use SVG for diagrams and icons when possible
