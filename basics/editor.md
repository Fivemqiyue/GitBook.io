# Editor

## Learn the basics of GitBook

The GitBook editor is designed to be intuitive and powerful. Here's what you need to know.

### Writing content

Simply click anywhere and start typing. The editor supports:

- **Bold** text with `**bold**`
- *Italic* text with `*italic*`
- `Code` with backticks
- [Links](https://gitbook.com)

### Adding blocks

Use the `/` command to add different types of content:

- `/heading` - Add headings
- `/code` - Add code blocks
- `/image` - Add images
- `/table` - Add tables
- `/hint` - Add hint boxes

### Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Bold |
| `Ctrl+I` | Italic |
| `Ctrl+K` | Add link |
| `Ctrl+/` | Toggle comment |
