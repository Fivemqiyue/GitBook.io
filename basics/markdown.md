# Markdown

## Writing with Markdown

GitBook supports standard Markdown syntax plus some extensions.

### Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
```

### Lists

```markdown
- Item 1
- Item 2
  - Nested item

1. First
2. Second
3. Third
```

### Code blocks

Use triple backticks with language identifier:

~~~markdown
```javascript
const hello = "world";
```
~~~

### Links and images

```markdown
[Link text](https://example.com)
![Alt text](image-url.png)
```

### Blockquotes

```markdown
> This is a quote
> It can span multiple lines
```

### Tables

```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
```
